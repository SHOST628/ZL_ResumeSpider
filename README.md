# ZL_ResumeSpider
智联简历筛选自动化工具

## 制作意图
本软件实际为各HR提供一个在智联招聘筛选简历时提供一个更加方便的工具。

## 编码环境
- python3.6
- requests
- PyQt5
- PIL
